TypeScript File System

Creating TypeScript file:

The TypeScript file extension is ".ts": [file-name].ts

Compiling TypeScript file:

$ tsc [file-name].ts
This creates a "[file-name].js" file in the same directory and we can use/run that file either in node or browser:
Running it in node:
$ node [file-name].js

VSCode Task

For easy developing and preventing of continues compiling of typescript files into JavaSript we can define a task in VSCode and just run it by "ctrl+shift+B" just once at the start of each time we run the VSCode.
For creating a task we need to follow below steps:
  1. Create a TypeScript project: tsc --init
  2. Go to VSCode command palet: ctrl+shift+p
  3. Type: configure task
  4. Choose "TypeScript watch mode" => this creates a "tasks.json" file
  5. Copy the below content into that file: {
    "version": "2.0.0",
    "tasks": [
    {
    "label": "My Task",
    "type": "shell",
    "command": "tsc",
    "args": [
    "-w", "-p", "../typescript_xstream"
    ],
    "isBackground": true,
    "problemMatcher": ["$tsc-watch"]
    }
    ]
    }
  6. Run the task by: ctrl+shift+b

TypeScript Basics

Variables:

Defining a Variable:
var [variable-name]:[variable-type] = [value];
Exmaples:
var myName: string = "ali";
var myAge: number = 30;
Variable types:
  • number:
    Double precision 64-bit floating point values. It can be used to represent both, integers and fractions.
    let myAge: number = 30;
  • string:
    let myName: string = "Ali";
    Template String: if we want to add many strings together it is easier to use template string:
    let greetings: string = 'Hello ${myName}';
  • boolean:
    let isAlive: boolean = true;
  • null:
    Represents an intentional absence of an object value.
  • undefined:
    Denotes value given to all uninitialized variables
  • any:
    This is the original variable type in JavaSript and if we don't define a variable type or if we expelictly define a variable as any it will be as type any. A variable which defined as any can change its type any where in the code:
    var testVariable;
    var testVariable: any;
    testVariable = 50;
    testVariable = "Hello"; // No error
  • void
  • Arrays
    let ages: number[] = [10,12, 8];
    let ages2 : Array = [15,16,18];

Static and Const:

A "static" variables (fields) belongs to the class itself and can be accessed using the class name:
static sval = 10;
"const" declarations are another way of declaring variables as their name implies, their value cannot be changed once they are bound:
const numLivesForCat = 9;

Conditional Operator (?)

The conditional operator is also sometimes referred to as the ternary operator. The syntax is as given below:
(conditional expression) ? [result-if-true] : [result-if-false];
EXAMPLE:
var num:number = -2;
var result = num > 0 ?"positive":"non-positive";

"typeof" and "instanceof" operators:

var num = 12;
console.log(typeof num); //output: number

class Person{ }
var obj = new Person();
var isPerson = obj instanceof Person;
console.log(" obj is an instance of Person " + isPerson);

Decision Making (if):

if
if ([boolean-condition]) { // statements; } else if ([boolean-condition-2]) { // statements-2; } else { // statements-3; }
switch
switch(variable_expression) { case constant_expr1: { //statements; break; } case constant_expr2: { //statements; break; } default: { //statements; break; } }

Loops:

for:
for (initial_count_value; termination-condition; step) { //statements }
Example:
for(i = num;i>=1;i--) { factorial *= i; }
for-in:
The for...in loop is used to iterate through a list or collection of values. The data type of val here should be string or any. For-in returns te index of that element in that list or tuple:
for (var val in list) { //statements }
Example
var j:any; var n:any = "a b c" for(j in n) { console.log(n[j]) }
OR
var n:any = "a b c" for( var j in n) { console.log(n[j]) }
for-of:
Unlike the for-in loop, for-of returns the actual values of each item in that list or tuple:
for(var val of n){ document.write(val + "
"); }
while:
while([condition]) { // statements if the condition is true }
do-while:
do { //statements } while([condition])

The Infinite Loops:

for (;;) {[statments]} while (true) {[statments]}

TypeScript Functions

Function definition:

First way
function [function-name]([arguments]): [return-type]{ // function body }
Example:
function getSum(num1: number, num2: number): number{ return num1 + num2; } consol.log(getSum(5, 4));
Second way
var [function-name] = function ([arguments]): [return-type]{ // function body }
Example:
var getSum = function(num1: number, num2: number): number{ return num1 + num2; } consol.log(getSum(5, 4));
