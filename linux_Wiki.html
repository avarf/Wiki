Install From Source

  1. cd [place of the downloaded program]
  2. makedi build
  3. cd build
  4. cmake
  5. make
  6. sudo make install

Note:

Dont delete the programs directory if you want to install it later instead do:
sudo make uninstall

Note:

Programs you install this way wont be automatically updated by Ubuntu’s Update Manager, even if they contain security vulnerabilities. Unless you require a specific application or version that isn’t in Ubuntu’s software repositories, it’s a good idea to stick with your distribution’s official packages.

    rm file_name
