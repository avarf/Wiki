Visual Studio Code - VSCode

Basic Editing in VSCode
Open command pallete: ctrl+p
Keybindings: ctrl+k + ctrl+s
Intellisense: For using Intellisense to see all the methods, classeswe, etc. we just need to go to command pallete and type "@" and it shows us all the methods.
Word Wrap: alt+z

Extensions:

Coding:

  • Numbered Bookmarks: (ctrl+p) ext install alefragnani.Bookmarks
    adding and removing bookmarks: ctrl+shift+[Number]
  • Find references to an interface: right click -- find references
  • Find peek: right click -- find peek
  • Paste JSON as Code: ext install quicktype.quicktype
    --Interactively generate types and (de-)serialization code from JSON, JSON Schema, and TypeScript
    --Paste JSON/JSON Schema/TypeScript as code
    Paste JSON as Code

Git:

We can use the git menue and just enter the commit message and press "ctrl+enter" for commiting the changes.

Python:

  • Python extension for Visual Studio Code: (ctrl+p) ext install ms-python.python
