Basic Editing in VSCode
Open command pallete: ctrl+p
Keybindings: ctrl+k + ctrl+s
Intellisense: For using Intellisense to see all the methods, classeswe, etc. we just need to go to command pallete and type "@" and it shows us all the methods.
Word Wrap: alt+z
Emmet: jusst write the main parts of the code and hit "tab" and the vscode will add the full code, for example: a.href -> "tab" ->
