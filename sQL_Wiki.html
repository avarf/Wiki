Crate Table

SQL table is a twodimensional arrayof rows and columns. We can create a table with "CREATE" command as below:
CREATE TABLE tableName
(
column_name_1 data_type,
column_name_2 data_type,
column_name_3 data_type
)

Example:


CREATE TABLE Persons
(
P_ID int,
LAstName varchar(255),
First_Name varchar(255),
Address varchar(255),
City varchar(255)
)

Simple Select

The simple method for getting a query from a table is SELECT command as below:
SELECT column_name(s) FROM table_name
SELECT * FROM table_name
Example:
SELECT Last_name, First_name FROM Persons
