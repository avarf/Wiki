ROPOD

  • Add or remove components to the robot definition such as adding camera so we don't see the error when we want to bringup the robot:

    - First we find the robot launch file
    - Inside the launch file we have to find the place that the code will save the robot model definition(package name, etc)
    - In that package we have to find the robot folder or similar folder
    - In that folder there is a file that defines the robot model such as: "youbot1_model.launch",etc.
    - We can add or remove components there
