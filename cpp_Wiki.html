Compile a C++ File In Linux

For compiling a .cpp file in linux we use:
g++ -std=c++11 [name_of_file.cpp] -o [name_of_output_file]

Notes:
  • The file can have as many as dependencies for classes and header files and call them internally and there is no problem.
  • For running the compiled file we run the name of the output file: [name_of_output_file.out] like this from the terminal: ./[name_of_output_file]
  • -std=c++11 defines that this files needs c++ 11 and we can use other versions too

Creating a CMake Package

For Cmake we first need a "CMakeLists.txt".

Video Tutorial:

Creatign "CMakeLists.txt"

Running Cmake on a project
  1. Create a "build" directory inside the directory that "CMakeLists.txt" exists: mkdir build
  2. Go to build: cd build
  3. Run cmake: cmake ..
  4. Run make for creating the executable file/files of our project: make
  5. Based on the content of our "CMakeLists.txt" then we run the executable file name in the "CMakeLists.txt": ./[file_address]/[executable_file_name]
